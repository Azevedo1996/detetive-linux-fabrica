# Treinamento — Detetive Linux

## Posicionamento

Este treinamento deve ser apresentado depois da Fantástica Fábrica do Linux.

- Fantástica Fábrica: conhecer comandos e conceitos.
- Detetive Linux: usar comandos para investigar situações práticas.

## Dinâmica sugerida

1. Relembrar a Fantástica Fábrica.
2. Explicar que agora o visitante virou detetive.
3. Jogar uma fase por bloco.
4. A cada acerto, reforçar a recompensa/evidência.
5. Encerrar com a Fase 5 e convite ao Detetive Linux 2.

## Mensagem final

Os comandos deixam de ser palavras soltas e viram ferramentas de investigação. O próximo passo é aprender a operar a fábrica em produção.
