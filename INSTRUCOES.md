# Instruções — Detetive Linux atualizado

## Objetivo

Alinhar o Detetive Linux como sequência natural da Fantástica Fábrica do Linux.

A Fantástica Fábrica ensina a perder o medo do terminal. O Detetive Linux usa esses comandos para resolver problemas em formato de investigação.

## Como carregar missões

```powershell
python manage.py migrate
python manage.py seed_missions
```

## Pontuação e recompensas

Cada missão vale pontos e entrega uma recompensa/evidência. A Fase 5 fecha a experiência e convida para o Detetive Linux 2.

## Fase final

A fase final revisa comandos essenciais e termina com uma tela de parabéns, encaminhando para a trilha intermediária.
